# 🤖 Task 1 – Machine Learning Classification Project

## 📌 Project Overview

This project was completed as **Task 1** of my **Artificial Intelligence Internship**.

The objective is to build and evaluate a supervised machine learning model that predicts customer churn using the IBM Telco Customer Churn dataset. Two classification algorithms were implemented and compared to identify the better-performing model.

---

## 🎯 Objective

- Build a supervised machine learning classification model.
- Compare Logistic Regression and Random Forest.
- Evaluate model performance using multiple classification metrics.
- Visualize model performance through plots.

---

## 📂 Dataset

**Dataset:** IBM Telco Customer Churn

**Target Variable:** Churn

The dataset contains customer information such as:

- Gender
- Senior Citizen
- Tenure
- Monthly Charges
- Total Charges
- Contract Type
- Internet Service
- Payment Method
- Churn Status
